# VR Integrations

IAPM works great on a standard desktop with keyboard and mouse. For a fully immersive experience, you can connect a VR headset.

## Supported Headsets

### [Meta Quest](Meta-Quest/index.md)

- Quest 2
- Quest 3
- Quest Pro

Connect via Meta Quest Link (wired or wireless).

### [HTC Vive](HTC-Vive/index.md)

- Vive
- Vive Pro
- Vive Pro 2

Connect via SteamVR.

## Requirements

- IAPM desktop client installed
- VR headset with PC connectivity
- Compatible controllers
- For wireless: strong Wi-Fi connection

## Getting Started

1. Install IAPM on your PC
2. Set up your VR headset following manufacturer instructions
3. Launch IAPM - it will automatically detect your headset
4. Put on your headset and enter the immersive environment

!!! info "New Integrations"
    We are continuously working on additional headset support. Check back for updates.
