# HTC Vive Integration

Connect your HTC Vive headset to experience IAPM in full VR.

## Supported Models

| Model | Status | Notes |
|-------|--------|-------|
| HTC Vive | :material-check: Supported | Full feature support |
| HTC Vive Pro | :material-check: Supported | Full feature support |
| HTC Vive Pro 2 | :material-check: Supported | Full feature support |

## Requirements

| Requirement | Details |
|-------------|---------|
| SteamVR | Installed and configured |
| Base Stations | 1.0 or 2.0 base stations set up |
| Controllers | Vive wand controllers |
| PC | Meets [VR requirements](../index.md#system-requirements) |

## Setup Guide

### Step 1: Configure SteamVR

1. Install [SteamVR](https://store.steampowered.com/app/250820/SteamVR/){ target="_blank" } from Steam
2. Set up your base stations following HTC's instructions
3. Complete SteamVR room setup
4. Test that your Vive is working in SteamVR Home

### Step 2: Launch IAPM

1. Start SteamVR (if not already running)
2. Launch IAPM Desktop on your PC
3. IAPM automatically detects your Vive headset
4. Put on your headset to enter the environment

### Step 3: Enter the Environment

Once connected, you'll enter your IAPM grid in full VR. Use your Vive controllers to navigate and interact.

## Controls

| Action | Control | Description |
|--------|---------|-------------|
| Teleport | Trackpad press + aim + release | Move to pointed location |
| Select | Trigger | Select service or UI element |
| Grab | Grip buttons | Grab and manipulate objects |
| Menu | Menu button | Open radial menu |
| AI Assistant | Grip + speak | Activate voice assistant |

See [Navigation & Controls](../../Guides/Navigation/index.md#htc-vive) for complete control reference.

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Headset not detected | Restart SteamVR, check cable connections |
| Tracking lost | Ensure base stations have clear line of sight |
| Controllers not tracking | Check battery levels, re-pair controllers |
| Poor performance | Lower graphics quality in IAPM Preferences |
| Gray screen | Check HDMI/DisplayPort connection |

## Tips for Best Experience

- **Position base stations high** - Mount above head height, angled down
- **Clear play area** - Remove reflective surfaces that can interfere with tracking
- **Check firmware** - Keep SteamVR and Vive firmware updated
- **Use wired audio** - For lowest latency audio experience
