# IAPM Web

Browser-based access to your application telemetry. No installation required.

[Launch IAPM Web (Entra ID)](https://azure.iapm.app){:class="md-button md-button--primary"}
[Launch IAPM Web (Local/Social)](https://my.iapm.app){:class="md-button"}

## What You Can Do

- **View Dashboards** - Monitor application health and performance metrics
- **Manage Grids** - Create, configure, and organize your telemetry data
- **Generate API Keys** - Get credentials to instrument your applications
- **Manage Your Account** - Handle subscriptions, billing, and team members

## Getting Started

1. Choose your authentication method and log in
2. Create or select a grid from your dashboard
3. Click **Enter** to view your application data

Need the full immersive experience? Check out [IAPM Desktop](../IAPM/index.md) for 3D visualization and VR support.
