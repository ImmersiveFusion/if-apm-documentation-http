# Browser Requirements

IAPM Web runs in any modern browser. No plugins or extensions required.

## Supported Browsers

| Browser | Supported Versions | Notes |
|---------|-------------------|-------|
| :material-google-chrome: Chrome | Latest 2 versions | Recommended |
| :material-microsoft-edge: Edge | Latest 2 versions | Full support |
| :material-firefox: Firefox | Latest + ESR | Full support |
| :material-apple-safari: Safari | Latest 2 versions | Full support |

## Mobile Browsers

| Platform | Browser | Supported Versions |
|----------|---------|-------------------|
| :material-apple: iOS | Safari | Latest 2 versions |
| :material-android: Android | Chrome | Latest 2 versions |

## Requirements

| Requirement | Details |
|-------------|---------|
| JavaScript | Must be enabled |
| Cookies | Required for authentication |
| Screen Width | Minimum 320px (responsive design) |
| Internet | Broadband connection recommended |

## Known Limitations

| Browser | Limitation |
|---------|------------|
| Internet Explorer | Not supported |
| Opera Mini | Limited functionality |

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Page won't load | Enable JavaScript, clear cache |
| Can't log in | Enable cookies, check browser version |
| Slow performance | Use Chrome or Edge, close unused tabs |
| Display issues | Update browser to latest version |
