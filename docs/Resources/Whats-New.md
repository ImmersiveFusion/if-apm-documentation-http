Recent updates and features in IAPM. For complete version history, see the release notes for [IAPM Desktop](../Products/IAPM-Desktop/release-notes.md) and [IAPM Web](../Products/IAPM-Web/release-notes.md).

## Latest Highlights

### AI Assistant (Desktop 1.11)

Interact with your telemetry using natural language. Ask questions via chat or voice - no queries required.

| Feature | Description |
|---------|-------------|
| **Chat interface** | Ask questions about your traces and metrics |
| **Voice commands** | Hands-free exploration with Tessa AI |
| **Always-listening mode** | Say "Hey Tessa" to activate |

### Span View & Network Graph (Web 3.71)

New visualization options for trace analysis:

| View | Best For |
|------|----------|
| **Span View** | Waterfall timeline showing span relationships |
| **Network Graph** | Interactive service dependency visualization |
| **Nested Tags** | Collapsible tree view for hierarchical data |

### macOS Support (Desktop 1.11)

Experimental support for Apple Silicon Macs. Download and test on macOS while we work toward full feature parity.
