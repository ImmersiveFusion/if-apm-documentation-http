# IAPM 1.12.0

This release brings the Skills system for extensible AI capabilities, multimodal vision support for pasting screenshots into prompts, and client parity with IAPM Studio for memory and context preservation.

## What's New
- Skills System: Extensible skill agent registry with catalog injection for modular AI capabilities.
- Multimodal Vision: Paste screenshots with Ctrl+V for visual prompts, allowing Tessa to analyze images alongside text.
- Composite Diagnostics Tools: New diagnostic and analysis tools for richer AI-assisted troubleshooting.
- Client parity with IAPM Studio: Memory.Local, Orchestration, and Context Preservation support.
- Migrated filter system to ADX data model.
- Updated AI Assistant to latest version.
