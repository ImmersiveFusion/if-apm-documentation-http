# IAPM 1.11.0

This release adopts the OpenTelemetry standard data model and refines the service graph with smoother transitions and improved placement.

## What's New
- Full OpenTelemetry-native data model for all trace and log visualizations.
- Smoother service graph startup with improved node placement.
- Tessa's tools updated with richer descriptions and better accuracy.
- Improved tooltips and stage labels.

## Fixes
- Fixed trace blocks occasionally stopping after data updates.
- Fixed service graph nodes not settling into stable positions.
