# IAPM 1.14.11

A series of stabilization patches across the 1.14.x line. Focused on Windows installer reliability, in-world polish, and an early preview of Boop that gets its full release in 1.15.0. Recommended 1.14.x build for everyone upgrading from 1.14.0.

## What's New
- Smaller, more reliable Windows installer that packages large worlds without hitting payload limits.
- Refreshed Windows installer dialog and banner artwork, with executable and shortcuts aligned to the DC naming convention.
- New macOS disk image distribution alongside the Windows installer.
- Boop preview with idle behaviors, props, and a contextual radial menu (disabled by default).
- Refined service facility content with overlay menu fixes, tooltip cleanup, and smarter error and log table rendering.
- Portal teleport landings that fall back to ground reliably, with portal rings facing the hub center.

## Fixes
- Fixed graph layout stability with improved bounding box handling and cooling behavior.
- Fixed avatar fall-through after portal teleport.
- Fixed prefab and scale issues across error and log tables.
- Fixed dependency tap-target resolution to use cell value instead of row index.
