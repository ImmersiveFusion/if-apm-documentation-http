## Getting Started

1. **Try the Sandbox** - Explore Immersive APM with pre-loaded sample data
2. **Create an Account** - Sign up via Azure Marketplace or directly
3. **Generate an API Key** - Get credentials for your applications
4. **Instrument Your App** - Add OpenTelemetry instrumentation to send telemetry data
