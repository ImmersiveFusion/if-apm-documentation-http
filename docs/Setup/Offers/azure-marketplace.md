# Azure Marketplace Listing

IAPM is listed on the Microsoft Azure Marketplace for visibility and discovery.

!!! info "Billing"
    IAPM is listed on the Azure Marketplace for discovery purposes. Billing and subscriptions are managed directly through IAPM at [portal.iapm.app](https://portal.iapm.app){ target="_blank" }.

## What the Listing Provides

| Benefit | Description |
|---------|-------------|
| **Discoverability** | Find IAPM through the Azure Marketplace catalog |
| **Enterprise login** | Sign in with Microsoft Entra ID (Azure AD) |
| **Trust signal** | Microsoft-vetted listing for procurement teams |

## How to Get Started

1. Visit the [IAPM Azure Marketplace listing](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/immersivefusion.iapm001){ target="_blank" }
2. Review the listing for product details
3. Sign up at [portal.iapm.app](https://portal.iapm.app){ target="_blank" } to create your account
4. Choose your [plan](../Plans/index.md) and subscribe
