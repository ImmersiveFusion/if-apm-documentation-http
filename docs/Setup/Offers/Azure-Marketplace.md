# Azure Marketplace Offer

Get IAPM through the Microsoft Azure Marketplace for simplified procurement and billing.

## Benefits

| Benefit | Description |
|---------|-------------|
| **Unified billing** | Charges appear on your existing Azure invoice |
| **Enterprise login** | Sign in with Microsoft Entra ID (Azure AD) |
| **Simplified procurement** | Use existing Microsoft purchasing agreements |
| **No credit card required** | Bill to your Azure subscription |

## What's Included

| Feature | Availability |
|---------|--------------|
| IAPM Desktop | :material-check: Full access |
| IAPM Web | :material-check: Full access |
| AI Assistant | :material-check: Included |
| Standard support | :material-check: Included |

## How to Subscribe

1. Visit the [IAPM Azure Marketplace listing](https://azuremarketplace.microsoft.com/en-us/marketplace/apps/immersivefusion.iapm001){ target="_blank" }
2. Click **Get It Now**
3. Sign in with your Microsoft account
4. Complete the subscription process
5. Access IAPM with your Entra ID credentials
