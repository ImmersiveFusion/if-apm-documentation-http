## Getting Started

1. **Install** - [Download IAPM Studio](Installation/index.md) for your operating system
2. **Launch** - Open IAPM Studio from your applications
3. **Sign in** - Authenticate with your IAPM account
4. **Connect** - Select a grid to begin querying telemetry data
5. **Chat** - Start a conversation with Tessa

## Using Tessa

### Chat

1. Type your question or request in the chat input field
2. Press Enter to send your message to Tessa
3. Review Tessa's response, including any tool results or data summaries
4. Follow up with additional questions to refine your investigation

### Voice

1. Click the microphone button to start voice input
2. Speak your question clearly
3. Tessa processes your speech and responds with text and spoken audio
4. Click the microphone button again to stop voice input

### Skills

1. Tessa automatically selects appropriate skills based on your request
2. Skills provide specialized analysis workflows for specific task types
3. Review the skill output in the conversation pane

### Multimodal Vision

1. Copy an image to your clipboard (screenshot, diagram, or dashboard capture)
2. Paste the image into the chat input field
3. Add a text prompt describing what you want Tessa to analyze
4. Tessa examines the image alongside your telemetry context

## Workspace Management

### Browsing Files

1. Open the workspace browser from the sidebar
2. Navigate the file tree to find workspace files and artifacts
3. Click a file to view its details

### Copying Paths

1. Right-click a file or folder in the workspace browser
2. Select the copy path option
3. Paste the path into a conversation or other application

## Multi-Tab Workflows

1. Open a new conversation tab from the tab bar
2. Start a separate investigation in the new tab
3. Drag a tab to split the view and see two conversations side by side
4. Resize panes by dragging the divider between them
5. Switch between tabs to manage multiple ongoing investigations
