# IAPM 1.12.0

This release introduces Skills for extensible AI capabilities, multimodal vision for analyzing screenshots, and feature parity with IAPM Studio.

## What's New
- Skills: Tessa can now use modular skills for specialized tasks.
- Multimodal Vision: Paste screenshots with Ctrl+V and ask Tessa to analyze what you see.
- Diagnostics Tools: New tools give Tessa richer troubleshooting insights.
- Feature parity with IAPM Studio for memory and context preservation.
- Improved filtering across all telemetry views.
